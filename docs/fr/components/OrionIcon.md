# OrionIcon

`<o-icon>` affiche une icône provenant de la bibliothèque [Material Icons](https://fonts.google.com/icons) avec la prop `icon`, ou de votre propre police d'icônes avec prop `font-icon`.

Vous pouvez facilement le personnaliser avec des props pour ajouter un effet de ripple au clic, un marqueur visuel pour indiquer des notifications, ou appliquer un style de bouton.

:::info
Au niveau de la configuration d'Orion, il est possible de paramétrer le style des icônes qui par défaut est `outlined`. ([plus d'informations ici](../guide/quick-start.md))
:::

:::tip
Le style du curseur passe automatiquement sur `pointer` si un écouteur `click` est utilisé sur `<o-icon>`.
:::
